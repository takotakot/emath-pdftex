showexample.sty は
    LaTeX のソースとタイプセットした結果
を対比して示すためのスタイルファイルです．

bs2yen.sty の ver.2 を使用できる環境では，
それを利用すると ＼ を ￥ で表示することが出来ます．

このファイルの在処は

  http://ms326.ms.u-tokyo.ac.jp/otobe/tex/package.html


使用例は同梱の sampleS.dvi (sampleS.tex) をご覧ください．

LaTeX2.09 でも使用可能ですが，ファイル名に8文字を超えるものが使えない場合は
ファイル名を showex.sty などと改名してください．
